API 文件

這裡列出了所有可用的 Static API 接口。由於是 GitHub Pages 託管，所有接口皆為 GET 唯讀請求。

GET/api/flex/template/:id.json

取得已渲染的 Flex Message JSON (Default)。

Parameters
  • :id - 樣板 ID
GET/api/flex/source/:id.txt

取得樣板原始碼 (Lodash Template)，可用於動態渲染。

Response Example
{
  "type": "bubble",
  "body": { "text": ${JSON.stringify(vcard.title)} } 
}
GET/api/flex/sample/:id.json

取得對應的資料欄位範本 (Sample Data)。

Response Example
{
  "title": "My Title",
  "date": "2024/01/01"
}
進階用法：動態渲染

本站不支援伺服器端渲染。若需動態帶入參數，請：

  1. 呼叫 Source API 取得 .txt 樣板字串。
  2. 呼叫 Sample API 確認可用欄位。
  3. 在您的程式中使用 lodash.template 結合資料進行渲染。
